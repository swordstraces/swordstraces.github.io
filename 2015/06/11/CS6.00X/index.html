MIT CS6.00复习笔记

很久之前就在简书上看到MIT这门课的推荐，由于时间一直对不上，之前都没能赶上EDX的最新在线更新，只是在MIT opencourse上或者网易公开课上看到09年和10年的旧版大学课堂录制视频。学习起来有些费劲，但是纵观课程syllabus，这门对于编程系统的入门还是相当不错的。本科结合python语言对计算机算法有一个初步的介绍。非常适合启蒙。最新的课程安排好像去掉了旧版中制图和数据处理的部分，将可能重点还是放在算法等核心思想的掌握上。

1.0 Lecture 1

What is computation?
  • what is knowledge?
  • declarative knowledge 陈述性知识
    • statements of fact
    • “the square root of a number x is a number y such that y*y = x”
  • imperative knowledge 祈使性知识
    • “how to” methods or recipes
    • “recipe” for deducing a square root:
      • start with a guess, called g
      • if g*g is close enough to x, stop and say that g is the answer
      • otherwise make a new guess, by averaging g and x/g
      • using this new guess, repeat the process until we get close enough
Aspects of languages
  • primitive constructs 原始结构
    • numbers, strings, operators
  • syntax - which strings of characters and symbols are well-formed
  • static semantics - which syntactically valid strings have a meaning 静态语义学，语法有错但是有意义
  • semantics - what is the meaning associated with a syntactically correct string of symbols with no static semantic errors

    错误原因

  • syntactic errors 语法错误
  • static semantic errors
  • 没有错但是不能完成目的
