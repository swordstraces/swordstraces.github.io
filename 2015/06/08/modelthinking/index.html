Model Thinking 模型思维

  • Model

模型思维是Coursera上密歇根大学的公开课程，概览syllabus觉得挺有意思，做一些笔记加深印象。

1.1 Reasons for model thinking

  1. Intelligent citizen of the world

    了解多种模型，并且学会具体应用

  2. Clearer Thinker

    例如一个人去餐厅吃饭，有多家餐厅选择，影响他的因素有：

    • 他有多少钱
    • 餐厅的消费水平
    • 他能够就餐的时间
    • 个人的口味偏好
  3. Understand and Use Data
  4. Decide, Strategize, and Design

1.2 Section Structure

  • The model
    • assumptions, results, applications
  • Technical Details
    • Measures, proofs(Easy, medium, hard)
    • practice problems
  • Fertility

1.4 Use and Understand Data

  1. understand patterns
  2. predict points
  3. produce bounds
  4. retrodict

  7. estimate hidden parameters
